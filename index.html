Repositórios GitHub

git-sistema-almoxarifado

Avaliação: Este repositório é voltado para o projeto de desenvolvimento de um sistema de gestão de almoxarifados e estoque. Inicialmente funcionou como um local de controle de versão, porém hoje é público para que outras desenvolvedores encontre falhas. A linguagem principal dos arquivos desse projeto é JAVA, e conta com api's de código aberto como Itext, PDFBox, BeansBinding, JCalendar e outras. Conta com uma base de dados desenvolvida em MySQL e hospedada em um servidor local em capelinha.

Atualizado em: 30/09/2021

cadmedicos-capelinha

Avaliação: Este repositório é voltado para um projeto de desenvolvimento de um sistema de gestão de laudos de alto custo, exames de tfd e exames AIH. O projeto visa diminuir o tempo e falhas ao preencher papeis dentro dos consultórios dos médicos. A linguagem predominante neste projeto é PHP, acompanhado de toda uma estrutura web em JavaScript, HTML e CSS. Conta com um sistema de login com usuários cadastrados e uma base de dados desenvolvida em MySQL, conta com um servidor na secretária de saúde de capelinha.

Atualizado em: 28/09/2021

git-portfolio-pessoal

Avaliação: Esse repositório é responsável por armazenar os arquivos de um projeto de portfólio pessoal. Usando técncias de CSS e JavaScript foi desenvolvida uma página de portfólio em modo "One-page-scroll" que basicamente é uma página única de HTML que contém todos os dados em um único local. Esse projeto foi desenvolvido visando a apresentação do meu trabalho como web Designer e técnico em informática. O projeto se encontra em fase final de desenvolvimento e já pode ser acessado através desse link.

Atualizado em: 01/10/2021
